# House Price Predictor

Application de prédiction des prix immobiliers utilisant un modèle Random Forest.

## Caractéristiques

- 7 features : surface, chambres, salles de bain, âge, terrain, garage, score quartier
- Visualisations interactives
- Intervalle de confiance à 95%

## Utilisation

Ajustez les sliders et cliquez sur "Prédire le prix".
